# Online Courses Analysis Dashboard (Power BI)

## Project Objective
To analyze online courses performance including pricing strategy, subscriber behavior, subject popularity, and yearly growth trends using Power BI.

## Dataset Used
- [Download Dataset](https://github.com/mahmoudnasr33/powerbi-online-courses-analysis/blob/main/udemy.csv)

## Questions (KPIs)

- What are the total number of courses, lectures, reviews, and subscribers?
- What is the average, minimum, and maximum course price?
- Which subjects generate the highest number of subscribers?
- How does pricing affect subscriber distribution?
- What percentage of courses are free vs paid?
- How did the number of courses grow over the years?

- Dashboard Interaction <a href="https://github.com/mahmoudnasr33/powerbi-online-courses-analysis/blob/main/overview.png">View Dashboard – Overview</a>
- Dashboard Interaction <a href="https://github.com/mahmoudnasr33/powerbi-online-courses-analysis/blob/main/price_analysis.png">View Dashboard – Price Analysis</a>

## Process

- Cleaned and transformed raw dataset using Power Query.
- Created calculated measures for pricing and subscriber metrics.
- Built interactive slicers and navigation buttons.
- Designed multi-page dashboard structure (Overview & Price Analysis).
- Applied data modeling for subject-level and price-level analysis.
