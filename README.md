# Online Courses Analysis Dashboard (Power BI)

## Project Objective
To analyze online courses performance including pricing strategy, subscriber behavior, subject popularity, and yearly growth trends using Power BI.

## Dataset Used
- [Download Dataset](https://github.com/mahmoudnasr33/powerbi-online-courses-analysis/blob/main/udemy.csv)

## Questions (KPIs)

- What are the total number of courses, lectures, reviews, and subscribers?
- What is the average, minimum, and maximum course price?
- Which subjects generate the highest number of subscribers?
- How does pricing affect subscriber distribution?
- What percentage of courses are free vs paid?
- How did the number of courses grow over the years?
